WELCOME TO THE WORLD OF SEGMENTATION


Welcome to Image Segmentation application. This application can be used to perform the following tasks.

  • Upload a image you want to do Segmentation .
  • The model will predict the segments in the image as different classes in COCO dataset .
  • It will give output, based on our Pre-trained Segmentation model.

    • Click on UPLOAD to upload an image .

    For complete usage of this application, view the rules defined below for various tasks.

    Rules for Uploading a Image

  • Data Set must be a image format jpg,jpeg,png....
  • One image can be uploaded once. Multiple images cannot be uploaded at a time.

    • Information regarding the Model Selected

  • In our case, we use the pretrained Deeplab network with Xception background
  • The model was trained on different classes on COCO dataset .
  • DeepLab is a state-of-art deep learning model for semantic image segmentation, where the goal is to assign semantic labels (e.g., person, dog, cat and so on) to every pixel in the input image.
